talent-aio 让天下没有难开发的即时通讯

简 介

talent-aio是基于java aio实现的即时通讯框架，功能类似netty和mina

应用场景

IM、实现各种网络应用层协议、实时监控、RPC等

特 点

  • 极易掌握: 无需各种折腾，只需花上30分钟学习helloworld就能很好地掌握并实现一个性能极好的即时通讯应用
  • 可同时支持10万级tcp长连接，彻底甩开业界当年的c10K烦恼
  • 收发200万条消息(约137M)，只需要1440毫秒(windows7、i7、8g、群聊场景)
  • 天生没有粘包问题
  • 框架层面帮你检测心跳(tcp server)、发送心跳(tcp client)
  • 可在框架层面帮你自动重连(tcp client)
  • 框架层面支持绑定userid(用于用户关联)、绑定groupid(用于群聊)

案 例

  • 某网管系统(管理数百台刀片服务器的系统)
  • 某直播平台(视频直播+聊天)
  • 某智能手表的数据采集系统
